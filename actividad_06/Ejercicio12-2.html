Background Color - color de Fordos

Si no especifica ningun color de fondo, el fondo es trasparente

CSS trata cada elemonto HTML Cono un cuadro y la propiedad background-color establece el color del fondo para el cuatro. se puede especificar un color de fonde de las mismas tres formas en el que se puede especificar colores del primer plano: valores RGB, codigo hexadecimal y nombre de colores. Tambien hemos utilizado la propiedad padding para separar el texto de los bordes de los cuadros. esto hace que sea mas facil leer.
